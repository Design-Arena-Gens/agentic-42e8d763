🤖 Agent - دستیار هوشمند شما

یک هوش مصنوعی قدرتمند برای کمک به برنامه‌نویسی و توسعه نرم‌افزار

Agent چیست؟

Agent یک دستیار هوش مصنوعی پیشرفته است که با استفاده از مدل Claude ساخته شده است. این ابزار می‌تواند به شما در طیف گسترده‌ای از وظایف برنامه‌نویسی و توسعه نرم‌افزار کمک کند - از نوشتن کد گرفته تا اشکال‌زدایی، طراحی سیستم، و مستندسازی.

قابلیت‌های کلیدی

💻 توسعه نرم‌افزار

  • نوشتن کد در زبان‌های مختلف
  • ایجاد پروژه‌های کامل
  • رفع باگ و اشکال‌زدایی
  • بهینه‌سازی کد
  • بازآرایی (Refactoring)

🔍 تحلیل و جستجو

  • جستجو در فایل‌های پروژه
  • تحلیل کد موجود
  • پیدا کردن الگوها
  • بررسی وابستگی‌ها
  • درک ساختار پروژه

🛠️ ابزارها و اتوماسیون

  • اجرای دستورات ترمینال
  • مدیریت Git
  • نصب وابستگی‌ها
  • اجرای تست‌ها
  • Deploy کردن پروژه‌ها

🌐 توسعه وب

  • ساخت وب‌سایت و اپلیکیشن
  • کار با React، Next.js، Vue
  • طراحی رابط کاربری
  • ساخت API
  • مدیریت پایگاه داده

📚 مستندسازی

  • نوشتن README
  • ایجاد مستندات API
  • کامنت‌گذاری کد
  • راهنمای استفاده
  • توضیح کد پیچیده

🎨 طراحی و UI/UX

  • طراحی رابط کاربری
  • استایل‌دهی با CSS
  • انیمیشن و افکت‌ها
  • طراحی واکنش‌گرا
  • بهبود تجربه کاربری

ساخت وب‌سایت کامل: "یک وب‌سایت فروشگاهی با React بساز که شامل سبد خرید، پرداخت و مدیریت محصولات باشد"
